Everything about Task Management

01

What views are included in the Work Module?

Board (Kanban), List View, My Tasks, Gantt Chart, and Calendar. Each view helps you plan and track work differently.

02

How does "My Tasks" differ from List View?

My Tasks shows only tasks assigned to you across projects, while List View displays all tasks with global filters & sorting.

03

Can I set dependencies in Gantt chart?

Yes. Link tasks (Finish-to-Start, Start-to-Start, etc.) and automatically adjust timeline when predecessors shift.

04

Is there a way to import existing tasks?

Absolutely. Import CSV/XLS files with columns: title, assignee, due date, priority, status — bulk create tasks.

05

Can I get reminders for upcoming tasks?

Yes. Email and in-app reminders before due date, daily digests, and overdue alerts.

06

Does the module support subtasks & checklists?

Yes. Create subtasks, checklists, attachments, comments, and time estimates for each task card.
